# Add a comment line to the input files

Used by the SS_write\* functions.

## Usage

``` r
writeComment(text, con, ...)
```

## Arguments

- text:

  Comment to write

- con:

  File to write to (passed to `con` input to
  [`writeLines()`](https://rdrr.io/r/base/writeLines.html))

- ...:

  Additional arguments passed to
  [`writeLines()`](https://rdrr.io/r/base/writeLines.html)
