# Get default vector of colors for each area

this was previously contained within SS_plots() and 4 of the SSplotXXX()
functions.

## Usage

``` r
get_areacols(areacols, nareas)
```

## Arguments

- areacols:

  Optional vector of colors for each area if model has multiple areas.
  NULL value will be replaced by a default set of areas.

- nareas:

  number of areas
