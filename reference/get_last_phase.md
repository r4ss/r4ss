# Get the highest phase used in the control file

Get the highest phase used in the control file

## Usage

``` r
get_last_phase(ctl)
```

## Arguments

- ctl:

  A control file list read in using
  [`r4ss::SS_readctl`](https://r4ss.github.io/r4ss/reference/SS_readctl.md).
